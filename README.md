# Event Hub

Event Hub is a demo backend project for event management, built with [NestJS](https://nestjs.com/) and TypeScript.  
It demonstrates how to structure a scalable, maintainable API for creating, managing, and publishing events, including draft support and user authentication.

## Features

- Event creation, update, and deletion
- Draft event management and publishing
- User authentication and role-based authorization
- RESTful API structure
- Prisma ORM integration
- Unit and E2E tests with Jest

## Getting Started

```bash
npm install
npm run start:dev
```

## Project Structure

- `src/` — Main source code (modules, controllers, services)
- `test/` — Unit and E2E tests
- `prisma/` — Prisma schema and migrations

## Testing

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
